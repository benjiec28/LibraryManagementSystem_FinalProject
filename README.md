# Library Management System

## Author : Benjamin Elli Cunanan

This is the final project for the course "DATA STRUCTURES AND OBJECT ORIENTED PROGRAMMING".
This project represents a library management system similar to those in real life. 
In it, we are able to borrow, return, add and remove items ; sort them with a certain field (like id or title) ; load and backup data ; etc.
This system is able to perform certain methods such as the following :  
    1) Validate Different fields (such as name, id, isbn and issue number).  
    2) Borrow and return items as well as searching for items, sorting them and check your available storage.
    3) Add and remove items from the library.
    4) Generate a report of items to verify certain fields and check their status.
    5) etc.  
Thanks to this, we are able to manage a replica of a library management system in a single program.
